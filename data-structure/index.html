Data structure

The goal is to understand when we use an array or an object and how to use them.

Array

Arrays are used to store a list of elements. The array prototype has methods which allows mutation operations (add, remove and access element ...). Arrays use integer indexes. Arrays are ordered lists.

Open your console and follow the instruction in the array.js script.

Object

Objects are used to store data with named properties. Object are used when you need to access data with identifiers.

Open your console and follow the instruction in the object.js script.
